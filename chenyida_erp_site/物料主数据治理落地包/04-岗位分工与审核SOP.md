# 04 岗位分工与审核 SOP

## 1. 岗位职责

| 角色 | 主要职责 | 不允许做的事 |
| --- | --- | --- |
| 采购 | 收集供应商物料表、价格、交期、MOQ，确认供应商映射 | 直接创建正式物料并投入 BOM |
| 工程 | 确认规格、封装、用途、替代关系、BOM 可用性 | 只按供应商名称判断物料相同 |
| 品质 | 确认环保、检验规则、供应商准入、特殊认证 | 放行未确认供应商或未定义检验规则的核心物料 |
| 仓库 | 确认单位、包装、批次、储位、有效期 | 使用不可换算的单位入库 |
| 财务/成本 | 确认计价单位、成本分类、税率 | 将样品价格直接作为量产成本标准 |
| 数据管理员 | 维护编码规则、合并重复物料、冻结异常物料 | 越权修改工程或品质结论 |

## 2. 新物料审核 SOP

1. 采购或工程提交新物料申请。
2. 系统先查重，输出自动匹配、疑似匹配、冲突匹配或新物料。
3. 疑似匹配优先绑定旧物料，不能轻易新建。
4. 确认为新物料后，工程补齐规格和用途。
5. 品质补齐环保、供应商准入和检验规则。
6. 仓库补齐单位、包装、批次和储位规则。
7. 财务确认成本分类和计价单位。
8. 数据管理员生成内部编码并启用。

## 3. 异常处理

| 异常 | 处理 |
| --- | --- |
| 重复建料 | 冻结新料，保留历史单据，后续用替代/合并关系处理 |
| 供应商料号冲突 | 暂停采购入库，采购和工程共同确认 |
| 客户专用料误用 | 冻结相关领料和 BOM，工程确认后再放行 |
| 单位不一致 | 仓库建立换算规则，不能换算则拆成不同物料 |
| 环保资料缺失 | 品质标记待补资料，核心物料不得放行量产 |
