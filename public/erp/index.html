今日状态

先把物料入口管住，再接 BOM、采购、库存和生产。

现场执行顺序

  1. 采购把供应商表贴到“供应商导入”。
  2. 系统自动匹配内部物料并分流。
  3. 自动匹配由采购确认，疑似匹配由工程确认。
  4. 新物料走建档，生成晨亿达内部物料编码。
  5. BOM、采购、库存、生产只使用内部物料编码。

品质处理原则

  1. IQC 负责来料检验和退供判断。
  2. IPQC 记录过程异常、责任工序和返工处置。
  3. FQC 确认成品出货前质量状态。
  4. 不良数量大于 0 时必须填写不良类型和处置方式。
