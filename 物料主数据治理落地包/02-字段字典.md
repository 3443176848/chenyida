# 02 字段字典

## 1. 内部标准物料库

| 字段 | 必填 | 说明 |
| --- | --- | --- |
| internal_item_code | 是 | 晨亿达内部物料编码 |
| item_category | 是 | 品类编码，如 CAP、RES、PI |
| standard_name | 是 | 内部标准名称 |
| item_status | 是 | 草稿、启用、停用、冻结 |
| base_uom | 是 | 基本库存单位 |
| brand | 否 | 品牌 |
| mpn | 否 | 原厂型号 |
| package | 否 | 封装，如 0402、0603、QFN |
| value_spec | 否 | 阻值、容值、电感值等 |
| voltage | 否 | 耐压 |
| tolerance | 否 | 精度 |
| material | 否 | 材质，如 X5R、PI、ED铜 |
| thickness | 否 | 厚度 |
| copper_weight | 否 | 铜厚 |
| environmental_level | 否 | RoHS、REACH、无卤等 |
| is_customer_specific | 是 | 是否客户专用 |
| default_inspection_rule | 否 | 默认检验规则 |
| remark | 否 | 备注 |

## 2. 供应商物料映射表

| 字段 | 必填 | 说明 |
| --- | --- | --- |
| internal_item_code | 是 | 对应内部物料编码 |
| supplier_name | 是 | 供应商名称 |
| supplier_item_name | 是 | 供应商原始名称 |
| supplier_item_code | 否 | 供应商料号 |
| supplier_brand | 否 | 供应商品牌 |
| supplier_mpn | 否 | 供应商提供的原厂型号 |
| purchase_uom | 是 | 采购单位 |
| uom_conversion | 否 | 采购单位到基本单位换算 |
| min_order_qty | 否 | 最小采购量 |
| lead_time_days | 否 | 交期天数 |
| last_price | 否 | 最近价格 |
| match_status | 是 | 已确认、疑似、冲突、停用 |
| approved_by | 否 | 确认人 |
| approved_date | 否 | 确认日期 |

## 3. 待清洗物料池

| 字段 | 必填 | 说明 |
| --- | --- | --- |
| import_batch_no | 是 | 导入批次 |
| supplier_name | 是 | 供应商 |
| raw_item_name | 是 | 供应商原始名称 |
| raw_item_code | 否 | 供应商原始料号 |
| raw_spec | 否 | 原始规格 |
| parsed_category | 否 | 系统解析品类 |
| parsed_package | 否 | 系统解析封装 |
| parsed_value | 否 | 系统解析规格值 |
| parsed_voltage | 否 | 系统解析耐压 |
| candidate_internal_code | 否 | 疑似内部物料 |
| match_level | 是 | 自动匹配、疑似匹配、冲突匹配、新物料 |
| confidence | 否 | 匹配置信度 |
| owner_role | 是 | 处理角色：采购、工程、品质、仓库 |
| process_status | 是 | 待处理、已确认、已建档、已驳回 |

## 4. 新物料申请审核表

| 字段 | 必填 | 说明 |
| --- | --- | --- |
| request_no | 是 | 申请单号 |
| request_source | 是 | 采购、BOM、客户图纸、工程变更 |
| proposed_category | 是 | 建议品类 |
| proposed_name | 是 | 建议名称 |
| supplier_name | 否 | 供应商 |
| supplier_item_code | 否 | 供应商料号 |
| engineering_result | 是 | 工程确认结果 |
| quality_result | 是 | 品质确认结果 |
| warehouse_result | 是 | 仓库确认结果 |
| finance_result | 否 | 财务/成本确认 |
| final_internal_code | 否 | 审核通过后生成 |
| final_status | 是 | 审核中、通过、驳回、补资料 |
