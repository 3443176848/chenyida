# 01 编码规则

## 1. 内部物料编码结构

建议格式：

```text
CYD-品类编码-6位流水号
```

示例：

```text
CYD-CAP-000001
CYD-RES-000001
CYD-PI-000001
CYD-CON-000001
```

编码原则：

- 编码只负责唯一识别，不承载过多规格信息。
- 规格、封装、品牌、环保、供应商等信息放在字段里维护。
- 物料编码一经启用不得复用。
- 停用物料只改状态，不删除历史记录。

## 2. 品类编码

| 品类编码 | 品类名称 | 示例 |
| --- | --- | --- |
| RES | 电阻 | 10K 1% 0603 |
| CAP | 电容 | 10uF 16V X5R 0402 |
| IND | 电感 | 2.2uH 0805 |
| IC | 芯片/IC | MCU、电源IC、接口IC |
| DIO | 二极管 | TVS、肖特基 |
| TRA | 三极管/MOS | MOSFET、BJT |
| CON | 连接器 | FPC连接器、BTB连接器 |
| PCB | PCB板材/加工件 | FR4、刚性板相关材料 |
| FPC | FPC成品/半成品 | 软板、软硬结合板 |
| PI | PI/铜箔/FPC基材 | PI、ED铜、RA铜 |
| AUX | 辅材 | 覆盖膜、胶纸、钢片、FR4补强 |
| PCK | 包装材料 | 吸塑盘、标签、纸箱 |
| SMT | SMT工艺材料 | 锡膏、红胶、钢网 |
| OTH | 其他 | 临时归类，必须定期清理 |

## 3. 标准名称规则

标准名称建议按“品类 + 核心规格 + 关键属性 + 封装/尺寸”组合。

示例：

| 品类 | 标准名称格式 |
| --- | --- |
| 电阻 | 贴片电阻 阻值 精度 封装 功率 |
| 电容 | 贴片电容 容值 耐压 材质 封装 精度 |
| 连接器 | FPC连接器 Pitch Pin数 高度 方向 |
| PI材料 | PI基材 厚度 铜厚 铜箔类型 胶系 |
| 覆盖膜 | 覆盖膜 膜厚/胶厚 宽度 材质 |

## 4. 单位规则

| 场景 | 建议单位 |
| --- | --- |
| 电子元件 | PCS |
| 料盘 | Reel，库存仍折算 PCS |
| PI/铜箔/覆盖膜 | M2 或 M |
| 板材 | PNL、SET 或 M2 |
| 锡膏/胶水 | KG、G、支 |
| 包装材料 | PCS、箱 |

单位必须有换算关系，禁止同一物料在库存中同时用多个不可换算单位。
