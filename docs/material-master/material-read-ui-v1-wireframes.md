布局规则：

- 筛选区默认紧凑展开，高度目标不超过约 104px。
- 不放总数、生效数或待审数统计卡片。
- 表格容器横向滚动；正式物料编码和标准名称为左侧固定列。
- 表头在表格容器内固定；分页紧邻表格，不使用遮挡内容的固定底栏。
- 操作列当前只有“查看详情”链接。
- 分类路径、长名称单行省略；悬停或键盘聚焦显示完整内容。

## 8. 错误状态

### 8.1 网络或服务端错误

```text
数据加载失败，请稍后重试
请求编号：3e0f…73e8（服务端提供时显示）
[重试]
```

### 8.2 401

```text
登录状态已失效，请重新登录
进入现有登录流程；成功后返回当前 URL
```

### 8.3 403

```text
当前账号没有查看此功能的权限
[返回 ERP 首页]
```

### 8.4 详情或历史 404

```text
物料不存在或无权查看
[返回物料列表]
```

不得区分对象不存在和行级不可见。

### 8.5 参数错误

```text
查询条件有误，请检查后重试
日期范围：起始日期不能晚于结束日期
[修正筛选] [重置]
```

## 9. 浏览器行为示例

```text
/materials?page=3&page_size=50&material_status=ACTIVE&sort=updated_at_desc
  -> 查看详情
/materials/456?return_to=<编码后的列表 URL>
  -> 版本历史
/materials/456/versions?page=2&page_size=20&return_to=<同一列表 URL>
  -> 浏览器后退
/materials/456?return_to=<同一列表 URL>
  -> 返回列表
/materials?page=3&page_size=50&material_status=ACTIVE&sort=updated_at_desc
```

刷新、复制 URL 和浏览器前进后退必须得到相同页面状态。

## 10. 禁止出现的视觉元素

- “新建物料”“编辑”“保存”“提交审核”“批准”“驳回”。
- Excel/CSV 导入入口。
- AI 分类、AI 建议、置信度或候选匹配入口。
- 版本恢复或日志删除。
- 用弹窗承载物料完整详情、完整版本历史或完整变更日志。
- 仅靠颜色表达状态、ERROR 或 WARNING。
